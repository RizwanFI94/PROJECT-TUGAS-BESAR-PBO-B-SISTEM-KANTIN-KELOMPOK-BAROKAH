##  Deskripsi Proyek
Sistem Kantin adalah aplikasi berbasis web yang dikembangkan untuk memenuhi Tugas Besar mata kuliah **Pemrograman Berorientasi Objek (PBO)**.  
Aplikasi ini bertujuan untuk membantu proses pengelolaan kantin, mulai dari manajemen pengguna, pengelolaan barang, transaksi pembelian, hingga pencetakan struk dalam bentuk PDF.

##  Fitur Aplikasi

###  Halaman Login
- Pengguna dapat login menggunakan username dan password.
- Sistem memverifikasi data login berdasarkan role pengguna.

###  Role Pengguna
Aplikasi memiliki dua role utama:
- **Admin**: Mengelola data barang dan data transaksi.
- **User**: Melihat produk dan melakukan pembelian.

###  Gambar Registrasi
- Halaman pendaftaran untuk pengguna baru.
- User wajib mengisi data sebelum dapat login ke sistem.

###  Halaman CRUD
- Admin dapat melakukan Create, Read, Update, dan Delete data barang.
- Data barang tersimpan di dalam sistem.

###  Tambah Barang
- Admin dapat menambahkan data barang baru.
- Informasi barang meliputi nama, harga, dan stok.

###  Edit Barang
- Admin dapat mengubah data barang yang sudah ada.
- Perubahan data akan langsung tersimpan.

###  Halaman Produk User
- User dapat melihat daftar produk yang tersedia.
- Menampilkan informasi produk secara detail.

###  List Keranjang
- User dapat menambahkan produk ke keranjang belanja.
- Keranjang menampilkan daftar barang yang akan dibeli.

###  Pop Up Struk Pembelian
- Sistem menampilkan struk pembelian setelah transaksi selesai.
- Struk berisi detail barang dan total pembayaran.

###  Cetak Struk ke PDF
- Struk pembelian dapat dicetak dan disimpan dalam format PDF.

### Melihat Daftar Data Transaksi
- Admin dapat melihat seluruh data transaksi yang terjadi.
- Data transaksi tersimpan sebagai arsip sistem.
